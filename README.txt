
#**************************   Prerequisites    #**************************

Install latest version of the node.

#**************************   Install npm modules and run webpack server   #*********************

Open Terminal and follow below steps.

1. Run command to install all npm modules given in package.json
   npm install
   
2. Build application.
   npm run build
  
3. Run webpack server
   npm start
   

#**************************   Open below link in browser   #**************************

http://localhost:9001/
